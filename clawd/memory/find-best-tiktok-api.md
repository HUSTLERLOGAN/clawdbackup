## **Testing Checklist**

Once you find a candidate API:

1. **Subscribe to free plan**
2. **Test these endpoints:**
   - Get user profile (check response time)
   - Get video details (verify data accuracy)
   - Get trending content (see if it's actually current)
3. **Check for errors:**
   - Rate limiting behavior
   - Error messages (are they helpful?)
   - Response consistency
4. **Time 10 requests:**
   - Average response time should be <2 seconds
5. **Compare data to TikTok app:**
   - Do follower counts match?
   - Are view counts accurate?
   - Is trending data current?
